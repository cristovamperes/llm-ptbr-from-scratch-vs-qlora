# Artefatos de análise

Esta pasta contém **saídas geradas** (JSON/TXT) usadas nas análises e no TCC.

- `samples/`: amostras geradas (prompts padronizados e variações).
- `results/`: métricas consolidadas (loss, PPL, etc.).
- `guardrails/`: métricas/checagens qualitativas (ex.: fallback ratio, short-piece ratio).
- `prompts/`: prompts e templates usados para geração/avaliação.
